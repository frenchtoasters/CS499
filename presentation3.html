Keeneland Concours d'Elegance Mobile Website

Nick Hollis, Shoba Dickinson, Blake Hall

Requirements:

  • A way to navigate the event
  • A way to disperse information to attendees

How: Mobile Website

Responsive (mobile-friendly) website with interactive map, help, and information.

Why a Mobile Website?

  • Easier to develop for more devices.
  • Don't have to worry about users updating an app.
  • No need for a sledgehammer when a regular hammer will do.
